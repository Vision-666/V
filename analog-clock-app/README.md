# Analog Clock Application

This project is an analog clock application built using PyQt5. It features a toggle switch that allows users to switch between dark and light themes.

## Features

- Displays an analog clock with hour, minute, and second hands.
- Dark and light theme toggle for user preference.
- Smooth updates every second to reflect the current time.

## Requirements

To run this application, you need to have Python installed along with the PyQt5 library. You can install the required dependencies using the following command:

```
pip install -r requirements.txt
```

## Running the Application

To run the application, execute the following command in your terminal:

```
python analog_clock.py
```

This will launch the analog clock window, where you can toggle between dark and light themes using the switch in the top-right corner.
